---
title: "Getting Started"
weight: 1
---

1. Clone the repository and create a local `.env` file from [`.env.example`](https://github.com/xian-technology/xian-intentkit/blob/main/.env.example).
2. Follow the [Local Setup](../deployment/local/) guide to start the stack from source.
3. Explore available endpoints at `http://localhost:8000/redoc`.
4. Create an agent configuration and interact with it via the API.
